# Convert classes from OWL model to JSON schema (in a very specific case yet)

## Install
```shell
pip3 install rdflib
```

## Usage examples
 - FOAF ontology:
```shell
python3 owl2jsonschema.py -s -f xml -u http://xmlns.com/foaf/spec/index.rdf 
```
